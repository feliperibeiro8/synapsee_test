# Projeto AI Engineer — Processo Seletivo SynapSee Studio

Este repositório contém a implementação de um projeto desenvolvido como parte do processo seletivo para estágio na SynapSee Studio.

O objetivo é demonstrar habilidades em desenvolvimento de sistemas com integração de modelos de Inteligência Artificial e proposta de solução para o problema apresentado.

## Estrutura do Projeto

* `backend/` — API responsável pela lógica e integração com o modelo
* `frontend/` — Interface simples para interação com o usuário
* `docs/` — Documentação e planejamento
